# REHA-public
SUMMARY
REHA is a web-based application designed to support physiotherapy assessment through an interactive body heatmap. It allows users to visually select areas of discomfort, define pain intensity, and record related symptoms.

The system provides a simple and intuitive interface for tracking physical conditions over time, helping both patients and physiotherapists monitor progress and improve decision-making.

REHA — Sistema de Mapa de Calor de Síntomas

Aplicación web interactiva orientada a la evaluación de síntomas musculares mediante un mapa corporal dinámico, diseñada para apoyar procesos de diagnóstico en fisioterapia.
🚀 Descripción

FisioMap permite a los usuarios identificar zonas de dolor en el cuerpo humano mediante una interfaz visual interactiva. A través de un sistema de selección por zonas, el usuario puede registrar:

Ubicación del dolor
Intensidad
Tipo de molestia
Síntomas asociados

La información se almacena para generar un historial clínico básico y facilitar el seguimiento del paciente.

🎯 Objetivo

Desarrollar una herramienta digital que:

Mejore la identificación de síntomas físicos
Facilite el seguimiento de pacientes
Apoye a fisioterapeutas en la toma de decisiones
Ofrezca una experiencia intuitiva y visual
🖥️ Características principales

✅ Mapa corporal interactivo (frente y espalda)
✅ Selección de zonas del cuerpo
✅ Sistema de intensidad de dolor (tipo heatmap)
✅ Registro de síntomas
✅ Historial de evaluaciones
✅ Interfaz amigable y responsiva

🧩 Tecnologías utilizadas
HTML5 — estructura del sistema
CSS3 — diseño y estilos
JavaScript — lógica e interacción
(Opcional) LocalStorage / Firebase — almacenamiento de datos
